# Raw

Staging area. Short-lived, not codified, **never load-bearing**.

A skill dumps research, scrape output, or pasted sessions here so the material
survives between Claude Code sessions. Then it either promotes the durable
parts into `wiki/` or abandons them.

- `research/` — web research, competitor findings, raw scrape output
- `chat-dumps/` — pasted Claude Code sessions worth keeping
- `inbox/` — anything not yet sorted

## Rules

- **Nothing here is a source of truth.** If a fact matters, it belongs in
  `wiki/` (codified) or `output/` (a finished deliverable). Code and skills
  must never depend on a file in `raw/`.
- **Disposable.** The monthly consolidation pass prunes stale `raw/` files. If
  something has not been promoted or used in a month, it can go.
- **Date your dumps.** `YYYY-MM-DD-<slug>.md` so staleness is visible.
