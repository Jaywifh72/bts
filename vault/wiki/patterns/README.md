# Patterns

Repeatable recipes that recur across CineCanon work. A pattern note explains
*how a kind of work is done here* so skills and humans don't re-derive it.

- [`entity-buildout.md`](entity-buildout.md) — the "new entity type" recipe:
  migration → schema → queries → routes → seed. Phases 1–4 all followed it.
- [`confidence-grading.md`](confidence-grading.md) — the T7-1…T7-7 rubric and
  what it controls (UI badge vs. ClaimReview structured data).
- [`plan-spec-workflow.md`](plan-spec-workflow.md) — how a feature goes from
  idea to merge: design spec → dated implementation plan → build → quality gate.

When a new repeatable recipe emerges, add a note here and link it from the
skill that uses it.
